# Stock Trading Platform (Zerodha Clone)

A full-stack stock trading platform inspired by Zerodha, built using React.js, Node.js, Express.js, and MongoDB. The platform enables users to manage stock holdings, track portfolio positions, monitor orders, and visualize investment performance through an interactive dashboard.

## Features

* User Authentication
* Portfolio Management
* Holdings Tracking
* Position Monitoring
* Order Management
* Interactive Dashboard
* Investment Performance Visualization
* Responsive User Interface

## Tech Stack

### Frontend

* React.js
* Material UI
* Chart.js
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Passport Authentication

## Installation

### Clone Repository

```bash
git clone https://github.com/Rishav060/stock-trading-platform.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Dashboard Setup

```bash
cd dashboard
npm install
npm start
```
